# Ruby Matrix Library

Библиотека для работы с матрицами. Реализует основные операции линейной алгебры: сложение, умножение, транспонирование, вычисление определителя, обратной матрицы

## Возможности

- Создание матриц разными способами
- Базовые операции: +, -, *, /
- Транспонирование
- Вычисление определителя (детерминанта)
- Нахождение обратной матрицы
- След матрицы
- Проверка свойств (квадратная, симметричная)
- Доступ к строкам и столбцам
